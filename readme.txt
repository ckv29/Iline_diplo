Система управления персоналом. Руководство.
Это простая система, где можно посмотреть список всех сотрудников, найти нужного и сменить руководителя — прямо на экране.

Flask-приложение для управления базой данных сотрудников с поддержкой сортировки, фильтрации, пагинации и изменения руководителей.

## Возможности

- Просмотр списка сотрудников с пагинацией
- Сортировка по любому полю (имя, должность, зарплата, дата найма)
- Фильтрация по:
 - Имени сотрудника
 - Должности
 - Имени руководителя
 - Диапазону зарплаты
 - Диапазону даты найма
- Поиск сотрудников по имени
- Изменение руководителя сотрудника
- Адаптивный интерфейс (2 колонки на десктопе)
- Сворачиваемые карточки сотрудников

## Требования

- Python 3.8+
- PostgreSQL 12+
- pip

##  API
## Endpoints

GET /
Главная страница. Перенаправляет на /sort_by/ с дефолтными параметрами.
GET /sort_by/
Список сотрудников с сортировкой, фильтрами.
Параметры:
field - поле сортировки: id, name, job_title, date_of_employment, salary, manager_id (по умолчанию: name)
order - порядок сортировки: asc, desc (по умолчанию: asc)
page - номер страницы (по умолчанию: 1)
per_page - количество записей на странице: 1-100 (по умолчанию: 40)
name - фильтр по имени (частичное совпадение)
job_title - фильтр по должности (частичное совпадение)
manager_name - фильтр по имени руководителя (частичное совпадение)
salary_from - минимальная зарплата
salary_to - максимальная зарплата
date_from - дата найма от (формат: YYYY-MM-DD)
date_to - дата найма до (формат: YYYY-MM-DD)

GET /find/<value>
Поиск сотрудников по имени.
Параметры:
value - строка для поиска (частичное совпадение, регистронезависимо)

POST /update_manager
Обновление руководителя сотрудника.
Параметры формы:
employee_id - ID сотрудника (обязательно)
new_manager_id - ID нового руководителя (пустая строка для удаления руководителя)
Особенности:
Сотрудник не может быть своим руководителем (кроме должности "CEO")
Если new_manager_id пустой, руководитель удаляется
При ошибке возвращается flash-сообщение
